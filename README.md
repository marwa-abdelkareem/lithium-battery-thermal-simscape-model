📌 Overview

This project presents a detailed electro-thermal lithium-ion battery cell model developed using
MATLAB, Simulink, and Simscape Electrical.

The model combines both electrical dynamics and thermal behavior of a lithium-ion battery cell, enabling accurate simulation of real-world operating conditions.

It is designed for applications in:

🔌 Electric Vehicles (EV)
🔋 Battery Management Systems (BMS)
⚡ Energy Storage Systems (ESS)
🎓 Academic research and simulations
🎯 Objectives
Develop an accurate electro-thermal battery cell model
Simulate State of Charge (SOC) behavior under dynamic loads
Analyze heat generation due to internal resistance
Study temperature rise and thermal effects
Support BMS algorithm testing and validation
⚙️ Modeling Environment
MATLAB R2024a
Simulink
Simscape Electrical
🔋 Model Features
⚡ Electrical Modeling
Equivalent circuit battery model
Internal resistance modeling
SOC estimation
Voltage response under load
🌡️ Thermal Modeling
Heat generation from electrical losses
Thermal mass representation
Heat transfer dynamics
Electro-thermal coupling
📊 Simulation Capabilities
Dynamic load testing
Temperature rise analysis
SOC tracking over time
Performance evaluation under different conditions

🔧 How to Run the Model
Clone the repository:
git clone https://github.com/marwa-abdelkareem/lithium-battery-thermal-simscape-model.git
Open MATLAB
Navigate to the project folder
Open the Simulink model:
batteryCellModelling.slx
Click Run ▶️ to start the simulation
